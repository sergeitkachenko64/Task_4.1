## Содержание:
### 1. Создание и настройка репозитория.
  + git init
  + git clone
  + git config

### 2. Сохранение изменений.
  - [git add](./add.md)
  - [git commit](./commit.md)
  - git diff
  - git stash
  - .gitignore

### 3. Проверка репозитория.
  * git status
  * git tag
  * git blame

### 4. Отмена коммитов и изменений.
  - git checkout
  - git clean
  - git revert
  - git reset
  - git rm

### 5. Переписывание истории.
  + git commit -amend
  + git rebase
  + git rebase -i
  + git reflog
